# book-used-trading
소장하고 있는 책을 편리하게 거래할 수 있는 서비스


## 시작하게된 계기

## 목표
- 객체지향적으로 유지보수가 편리하고 읽기 좋은 코드 작성
- 채팅 서비스
  - 실시간
  - 1대1 대화
- 쪽지

## 요구사항 분석
- 책을 판매하고 싶은 회원이 이미지와 함께 판매 글을 등록할 수 있다.
  - CRUD
- 구매하고 싶은 회원은 판매자와 1대1 대화를 통해 거래할 수 있다.
- 실시간 또는 메시지(쪽지)를 보낼 수 있다.

## 도메인 모델
